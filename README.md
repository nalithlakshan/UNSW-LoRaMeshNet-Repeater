This repository contains the hardware and firmware artifacts for the **UNSW-LoRaMeshNet Repeater**, a multi-radio LoRa repeater designed for energy-efficient multi-hop LoRa mesh networking as part of the UNSW LoRaMeshNet.

The repeater hardware is based on four RAK3172 modules (STM32WLE5) and is intended for LoRa mesh network deployments, particularly in challenging environments such as underground mines and tunnels.

## Publications

The development of the **UNSW-LoRaMeshNet** platform was motivated by the following research work:

1. N. Udugampola, X. Ai, B. Li, and A. Seneviratne, “Scalability analysis of linear LoRa mesh networks,” in *Proc. IEEE 32nd Int. Conf. Modeling, Analysis and Simulation of Computer and Telecommunication Systems (MASCOTS)*, 2024.

2. N. Udugampola, X. Ai, B. Li, H. Gong, and A. Seneviratne, “A position-and energy-aware routing strategy for subterranean LoRa mesh networks,” in *Proc. IEEE 50th Conf. Local Computer Networks (LCN)*, 2025.

3. N. Udugampola, X. Ai, B. Li, H. Gong, and A. Seneviratne, “LoRaMSim: A simulation platform for advanced LoRa mesh networks,” submitted for review.

4. N. Udugampola, X. Ai, B. Li, H. Gong, and A. Seneviratne, “Adapting the LoRaWAN Relay Specification for battery-sustainable subterranean LoRa mesh networks,” submitted for review.
